# btreeprint
a basic package that allows printing binary search trees horizontally
Example:


```
Before:
01
└──07
   ├──12
   │  └──10
   └──05
      └──02
         └──03
__________________

After:

01
└──07
   ├──55
   │  ├──60
   │  └──33
   │     └──12
   │        └──15
   └──05
      └──02
         └──03
